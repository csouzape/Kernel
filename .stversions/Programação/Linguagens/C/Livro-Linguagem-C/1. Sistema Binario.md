#C
Qualquer coisa processada por um computador sejando celular, tablet ou qualquer dispositivo digital opera com *aritmerica  binaria* **(processando informações atravez de numeros, principalmente 0 e 1)**

Em quanto a aritmetica usa algarismos de 0 a 9, a logica binaria usa apenas dois valores, 0 e 1
sendo 0 = falso ou desligado, e 1 = verdadeiro ou ligado 

## 1.1 - Conversoes de base (Decimal)
Decompor numeros decimais: o numero 123 e composto por 3 = unidadade, 2 = dezena, 1 = centena 

para decompor 123 seria = $1*100+2*10+3*1=123$

## 1.2 - Conversoes de base (Binaria)
Decompor o numero binario: 101 é decompondo em $1*2²+0*2¹+1*2⁰$
outro exemplo: 10101 = $1*2⁴+ 0*2^3+1*2²+0*2¹+1*2⁰$
= 21
